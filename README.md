# Electron-clock

## Requirements
 - Node.js 10.x, 11.x
 - electron
 - electron-builder

## Installation
1. Clone this repository.
2. Run `npm install`.
3. Run `npm install electron --save-dev`.
4. Run `npm install electron-builder --save-dev`.
5. Run `npm run pack:osx`.
